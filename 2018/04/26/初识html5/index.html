初识html5

html5是什么

万维网的核心语言、标准通用标记语言下的一个应用超文本标记语言（HTML）的第五次重大修改（这是一项推荐标准、外语原文：W3C Recommendation ）。

不兼容IE8及以下
doctype 声明文档类型
标签可以大小写 但规范为小写
class id 名等等可以不需要加引号 但规范为加
h5标签具有语义化 浏览器认识
header 头部
section 大区块
footer 底部
<div class="head"> </div>并没有语义化 浏览器并不认识

nav 导航
article 文章 代表一个完整的相关内容快,一般是用在文章类

如果article嵌套使用,里面acticle内容一定要与外面的内容相关联
aside 非正文内容 一般用在侧边栏以及广告位
hgroup 主标题和副标题的组合 两人及以上的组合
figure 对图片,视频等多媒体标签组合
figcaption figure组合的标题等
time 事件
datalist 数据列表 通过id和input的list属性绑定

details 摘要 默认为折叠状态

  • open 设置默认为展开状态

dialog 对话框 默认位置在网页中部,不显示

adress 地址
mark 标记 文字高亮 像做笔记用颜色笔涂抹一样
meter 测量 刻度条

  • value 设置尺度的长度
  • max 设置尺度的最大值
  • min 设置尺度的最小值
  • high 设置尺度的最高值 低于为绿色 高于为黄色
  • low 设置尺度的最低值 低于为黄色 高于为绿色
    设置任何样式不生效
    progress 进度条
  • value 定义进程的当前值
  • max 定义完成的值

html5 布局

html5 兼容处理
引入html5shiv.js
