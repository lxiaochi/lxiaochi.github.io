jQuery之筛选

发表于 2018-04-07 | 分类于 JQuery
字数统计: 521 | 阅读时长 ≈ 2

筛选

1. eq()———-下标

:eq() 选择器选取带有指定 index 值的元素。
index 值从 0 开始，所有第一个元素的 index 值是 0（不是 1）。

2. index()———-索引

index() 方法返回指定元素相对于其他指定元素的 index 位置。
这些元素可通过 jQuery 选择器或 DOM 元素来指定。
注释：如果未找到元素，index() 将返回 -1。

3. filter(selector)———-有

一个现有的jQuery对象，用于匹配当前的一组元素。

4. not(selector)———-没有

:not(selector) 选择器选取除了指定元素以外的所有元素。

5. first()———-第一个

first() 将匹配元素集合缩减为集合中的第一个元素。

6. last()———-最后一个

last() 将匹配元素集合缩减为集合中的最后一个元素。

7. has()———-子元素包含

has() 将匹配元素集合缩减为拥有匹配指定选择器或 DOM 元素的后代的子集。

8. add( selector)———-添加JQ对象

add() 方法将元素添加到匹配元素的集合中。

9. next()———-上一个兄弟

next() 获得匹配元素集合中每个元素紧邻的同胞元素。如果提供选择器，则取回匹配该选择器的下一个同胞元素。

10. prev()———-下一个兄弟

prev() 获得匹配元素集合中每个元素紧邻的前一个同胞元素，通过选择器进行筛选是可选的。

11. siblings([selector])———-除自身外的兄弟节点

siblings() 获得匹配集合中每个元素的同胞，通过选择器进行筛选是可选的。

12. children()———-子元素

children() 方法返回返回被选元素的所有直接子元素。

13. find()———-查找指定子元素

find() 方法获得当前元素集合中每个元素的后代，通过选择器、jQuery 对象或元素来筛选。
