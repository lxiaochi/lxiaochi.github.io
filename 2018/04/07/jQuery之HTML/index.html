HTML

1、class操作

hasClass()-————————–是否包含某一个class，不能用函数
addClass() —————-添加一个或多个类

removeClass() ———- 删除一个或多个类
toggleClass() ————添加/删除类的切换操作

2、设置/获取 元素内容

text() - 设置或返回所选元素的文本内容(不包含HTML标签 相当于innerTest)
html() - 设置集合第一或返回 所选所有元素的内容（包括 HTML 标签 相当于innerHTML）
val() - 设置或返回表单字段的值

3、HTML属性

attr()————–设置/获取属性

  1. 一个参数：获取属性
  2. 两个参数：设置属性
  3. 一个json格式的参数：设置一堆属性
  4. 可以设置自定义属性

prop()与attr()相似,但不能设置自定义属性

removeAttr()————–设置/获取属性

  • data() 获取/设置 元素数据—-不可传递函数
  • removeData() 移除数据

4、盒模型

width() ———————获取/设置宽度

border-box 下，获取的（包含padding border），设置的仅仅content的值 （不包含padding border）

innerWidth()—————获取宽度+padding
outerWidth()————–获取宽度+padding+border
outerWidth( true)————–获取宽度+padding+border+margin

5、位置

  1. offset()

    以文档为参考来计算位置，是一个对象，带有top,left值 。相当于是固定定位
    offset().top
    offset().left

设置

  1. parent() —父级
  2. offsetParent() —定位父级

  3. position() —–到定位父级的偏移量

    position()也是一个对象，包含top,left。是到定位父级的偏移量—–只读

  4. scrollTop() / scrollLeft()— 设置/获取 元素滚动条位置

    可以用于window,document上，不传递函数
