jQuery $()

发表于 2018-04-06 | 分类于 JQuery
字数统计: 470 | 阅读时长 ≈ 2

一、jQuery函数

jQuery另一个名称$,JQuery()等同于$()

jQuery设计精髓

  • 强大的函数式编程
  • 链式操作
  • getter setter是合体的
    getter是获取中集合中的一个的值，setter是设置集合中的每一个

二、$()的四大调用方式

1. $(function(){})

文档加载完毕执行函数,等同于DOMContentLoaded,其他写法$(document).ready(function(){})

2. $(selector,start)

selector 是一个CSS选择器,start 可选 在该元素下或子元素下面去匹配

length长度

jQuery对象中元素的个数
这个函数的返回值与jQuery对象的size()属性一致

  • css()获取/设置样式

    css(key,val) 参数形式

    1. 当css("")的参数只有一个时,表示获取元素
    2. 当css("","")的参数有两个时,表示获取元素,并设置此元素的属性
    3. 当css({ key,value})的参数为对象时,使用键值对形式可以添加多个css样式
    • 可以使用连字符 :background-color 驼峰名属性:backgroundColor
    • 获取计算后的css样式
    • 不能获取复合样式
    • 遵循css样式优先级,用css()添加的样式为行内样式,并且会将元素原来的行内样式覆盖
    • value值可以是函数,可以使用return函数

  • click()点击事件

    click( function(){} ) 参数是一个函数

  • selector属性

    创建jQuery对象时 css选择器字符串

  • jQuery()JQuery版本号

    所有JQuery对象下都有JQuery属性

3. $( element )

传递给一个Element,document,window 会把对象转为jQuery对象

4. $( tag,object )

tag创建元素，是字符串标签，得带上<>括号
object 是一个对象，对象的属性可以是csss,html,text,width,height,offset,val或data,或者是事件类型
