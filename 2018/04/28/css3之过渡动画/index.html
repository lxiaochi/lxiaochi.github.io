css3之过渡动画

发表于 2018-04-28 | 分类于 html5
字数统计: 532 | 阅读时长 ≈ 2

transition过渡

transition: property duration timing-function delay;
从一种样式逐渐变成另外另一种样式

  • transition-property 指定过渡属性的名称,默认是all,全部的意思
  • transition-duration 指定过渡效果需要用的时间
  • transition-timing-function 设置过渡效果的速度
    ease 默认值,开始慢速,然后加快,最后慢速结束
    linear 匀速运动
    ease-in 开始慢速,之后一直快速
    ease-out 开始快速,之后一直慢速
    ease-in-out 开始慢速,然后加快,最后慢速结束
  • transition-delay 延迟执行时间

animation 动画

animation: name duration timing-function delay iteration-count direction;
动画帧

  • animation-name 指定动画的名称
  • animation-duration 指定动画播放1次的时间
  • animation-iteration-count: 指定动画播放的次数
    num 自定义次数
    infinite 无限循环
  • animation-timing-function 设置动画效果的速度
  • animation-delay 延迟执行时间
  • animation-direction 设置动画的执行方向
    normal 正常播放,默认值
    reverse 动画方向播放
    alternate 动画在奇数次正常播放,在偶数次反向播放
    alternate-reverse 动画在奇数次反向播放,在偶数次正常播放
  • animation-play-state 设置动画是否暂停或者运行
    paused 暂停动画
    running 开始动画
  • animation-fill-mode
    forwards 在动画结束后停在动画帧100%的位置
    backwords 忽略原始状态,直接进入动画0%的位置
    both 以上两者的结合
    关键帧 动画名称
    @keyframes name
    from 动画帧开始
    to 动画帧结束

    新增 颜色值和文本阴影

  • rgba
    r:红色 0-255
    g:绿色 0-255
    b:蓝色 0-255
    a:透明度 0-1
  • hsla
    h:色调 0/360表示红色,120表示绿色,240表示蓝色,0-360
    s:饱和度 0%-100%,0%灰色阴影色,100%全彩色
    l:明度 变暗—50%(正常纯色)—100%白色
    a:透明度 0-1
  • text-shadow 属性向文本设置阴影。
    text-shadow: h-shadow v-shadow blur color
    h-shadow 必需。水平阴影的位置。允许负值
    v-shadow 必需。垂直阴影的位置。允许负值
    blur 可选。模糊的距离
    color 可选。阴影的颜色
