数据类型

常量与变量

标识符的命名规则:

  1. 不能使用系统关键字
  2. 允许使用字母数字下划线
  3. 不允许数字开头
  4. 标识符区分大小写
  5. 见名知义 驼峰命名法
    常量: 在程序运行过程中 其值不能发生改变的量 称为常量
    定义常量方式:
    const 数据类型 常量名 = 值 (不安全)
    #define 常量名 值

变量: 在程序运行的过程中 其值可以发生改变的量 称为变量
定义变量方式:
数据类型 变量 = 值
变量 = 表达式
在定义局部变量可以在数据类型前加auto修饰符 也可以不加

关键字

C的关键字共有32个

  • 数据类型关键字(12个)
    char,short,int,long,float,double,
    unsigned,signed,struct,union,enum,void
  • 控制语句关键字(12个)
    if,else,switch,case,default,
    for,do,while,break,continue,goto,return
  • 存储类关键字(5个)
    auto,extern,register,static,const
  • 其他关键字(3个)
    sizeof,typeof,volatile

    数据类型

    数据类型的作用:编译器预算对象(变量)分配的内存空间大小 定义八进制数据 以0开头
    int a = 0123
    定义十六进制数据 以0x开头
    int b = 0x123
    在计算机定义数据时,不可以直接定义二进制

scanf安全问题解决:
#define _CRT_SECURE_NO_WARNINGS
#pragma warning(disable:4496)

sizeof 计算数据类型在内存中占的字节(BYTE)大小
sizeof(数据类型) sizeof(变量名) sizeof 变量名 不建议使用
