变量:

本质上内存中的一小块区域,他是可以在程序运行中 在指定范围内发生变化的量 叫做变量
注意事项:
1:要有数据限定的类型
2:给他起个名字
3:给他一个初始化值(就是在定义一个变量的时候给他赋值)
定义变量格式:
数据类型 变量名 = 初始化值;

定义变量:

格式:
数据类型 变量名 = 初始化值;
先声明 后赋值
数据类型 变量名;
变量名 = 初始化值;
byte short int long float double boolean char
注意:
int 定义整数的时候默认都是int类型 如果要把一个值赋值给long类型的变量 那么需要指明他的类型为long 在数值后面加L
float 定义浮点型的时候默认都是double 如果要把一个浮点值赋值float 那么需要指明他的类型为float 在数值后面加f/F
定义变量注意事项:
1:定义的变量如果要使用必须先赋值
2:作用域 范围 定义的变量只在他所属的范围内有效
3:一行上可以定义多个变量 但是不建议这么使用
